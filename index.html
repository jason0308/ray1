學習環境及資源

本系目前擁有一座系館，2016年更打造一間物聯網創客基地，提供給學生創意發想的環境，加上14位畢業於國內外名校的頂尖師資，給你最佳的學習環境與研究資源！

Icon

密切的師生關係

完善的「導師制度」：本系大一各班安排雙導師，其餘年級各班安排一位導師，透過導師制度的關懷與教導，讓學生能瞭解大學生活該有的態度，或在學生遭遇了某些困擾，能有導師給予較適切的指引。

Icon

最具前瞻性的科技

本系的研究方向與課程規劃永遠走在時代的尖端。

Icon

五年讀完學碩士

對於成績優異的同學，本校提供「五年學碩士學位取得辦法」，讓有心及早就業或出國深造的同學有更多的選擇。

Icon

精彩的大學生活

本系的學生在中原十分活躍，無論是新生盃的運動競賽、各式各樣的校隊與社團活動，皆能看到資管人的蹤跡，可說是「能讀又能玩」。

Icon

強大的就業資源

本系畢業系友，發展方向相當廣泛並在各領域都有傑出表現。傑出系友如：台灣微軟公共業務事業群總經理、雨傘王及手機王創辦人、巨網資訊有限公司 (Paint人集團、網路溫度計)創辦人、VoiceTube 創辦人等。
